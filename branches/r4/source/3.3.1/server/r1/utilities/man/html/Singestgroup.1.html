NAME

Singestgroup - adds a group user account to a SRB grid.  

SYNOPSIS

Singestgroup NewGroupName NewGroupPassword NewGroupAddress NewGroupPhone NewGroupEmail  

DESCRIPTION

Singestgroup adds a group account to a SRB grid. Be careful to quote fields with spaces.

 

FIELDS

NewGroupName
the name for the group
NewGroupPassword
the account password
NewGroupAddress
the address of the group
NewGroupPhone
the phone number of the group
NewGroupEmail
the email of the group
