NAME

Serror - describes SRB errors.  

SYNOPSIS

Serror number  

DESCRIPTION

Serror takes an error number generated by SRB/MCAT system and displays a message relating to the error.

 

OPTIONS

none  

BUGS

Not all error numbers are processed.
