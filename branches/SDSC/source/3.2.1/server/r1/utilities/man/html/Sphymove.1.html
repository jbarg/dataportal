NAME

Sphymove - moves an SRB object to a new resource  

SYNOPSIS

Sphymove [-C n] [-p] [-P newpathName] srbObj ... newresourceName  

DESCRIPTION

Sphymove moves one or more SRB objects to the newresourceName at new path newpathName (if given). The old copy is deleted and the MCAT catalog is also updated.

If given, the object is stored in newpathName. Otherwise the object is stored in the default path for the newresourceName stored in MCAT.

The user should have at least 'write' access permission for srbObj.

 

OPTIONS

-h
display command options
-p
prompts before moving each object to SRB space.
-C
n is an integer denoting the COPY of the object to be copied.
-P
full or relative newpathName to move the object.
