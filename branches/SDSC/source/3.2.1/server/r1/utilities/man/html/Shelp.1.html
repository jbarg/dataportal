NAME

Shelp - Displays a list of all available SRB S-commands  

SYNOPSIS

Shelp  

DESCRIPTION

Shelp Displays a list of all available SRB S-commands, as well as a short description of what it's for.

 

OPTIONS

none

Sinit(1), Senv(1), Sexit(1)
