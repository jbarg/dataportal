Loading Certificate into MyProxy server

Certificates uploaded into a MyProxy server have only a limited lifetime for security reasons, therefore you should make sure that yours is still valid. If in doubt just re-upload your certificate it will replace any existing certificates. When up loading your certificate you can decide, which lifetime it should have. If you would like to use CCLRC's MyProxy server you can upload your certificate by going to the login page of the DataPortal and click on e-Minerals Proxy. This will open the DataPortal Proxy tool.
Setting up Cog Kit

This process usually only needs to be carried out when you first use the upload tool, please keep the resulting file in a directory where it can not be removed. If you have installed the Cog Kit before please proceed to the next step 'Create Proxy'. Otherwise follow the following steps again.

Please click on 'Config' and select 'setup cog', this will open the Java Cog Kit configuration wizard. After reading the welcome page please click on 'Next' to progress the installation.

In a first step you define where your user credentials - User Certificate and User Private Key - are (see previous section 'What to do before you use the Grid for the first time' for more details on how to create these). If you have used the upload tool before hand this will already have the correct position of your files and you can proceed straight away to the next step. Otherwise you can either type the correct path into the fields below the names or click on the '..' button to use the browsing facility to choose the correct file location (incl. file name). When finished please select 'Next' to go on to the next step.

The next step allows you to download the certificate of your Certification Authority (CA) and register it with the download tool, if you have used the tool before you can proceed immediately to the next step by pressing 'Next'. Otherwise please click on 'Download UK e-Science Certificate' if you have a certificate authorised by the UK e-Science Community. You will then be asked if you want to download and install the UK e-Science certificate, please press 'OK' to proceed. You will than see a window that informs you of the successful download of the certificate giving location and name for the certificate, please press 'OK' to proceed. Another window called 'Add CA Certificate' will then open and you should choose the certificate that you just downloaded by clicking on its name in the file list of the window. Once done please press 'Select' to proceed. You will than go back to the Configuring Certification Authorities (CA) page. If you are using another certification authority than the UK e-Science one please click on 'Add' to add their certificate to the download tool list. This will open another window, which will allow you to browse your computer for the correct Certification Authority Certificate, once found please select it by clicking on the file name and press the 'Select' button to add it to the download tools list of CA certificates. You will than go back to the Configuring Certification Authorities (CA) page. Please press 'Next' to proceed to the next step.

The tool has now all the information it need's to create a temporary proxy file. The name and the location where the tool intends to place it will be shown to you on the Configuring Proxy File web page. Please check that you have write access to the chosen location, otherwise click on '..' to chose a more appropriate name or location. Press 'Select' to confirm you new choice, this will take you back to the Configuring Proxy File page. Now click on 'Finish' to complete the installation of the Java Cog Kit. A little window will appear to confirm the successful installation, please press 'OK' to proceed. This will take you back to the e-Minerals Proxy Tool.
Creating Proxy

Please select 'Config' and press on 'Create Proxy' to create your proxy certificate, this will open the Grid Proxy Init page. Here you will be asked for your password (the one you used to create your certificate in the first place), please type it in and click on 'Options' to select the lifetime of your certificate (max 1 months). Please select 'Apply' to return to the previous page. Please press 'Create' to create your proxy certificate a little window will show you the progress of this process, followed by a window to let you know the process is finished. Please press 'OK' to return to the main page.
Upload Proxy Certificate

Finally you can now upload your proxy certificate, which you will be able to use for the time period chosen in the 'create proxy' step. Please chose 'Config' and select 'Send to MyProxy', this will open the MyProxy configuration window. The window will show all the information necessary to upload the certificate, please do not change. There please press 'Send' to send your proxy certificate. Next the Details page will open, where you will be asked for your user name (please use the one you registered with at the e-Minerals MiniGrid) and your chosen passphrase. Please fill in the details and press 'OK' to complete the upload process. A little window will show you the progress of this operation, followed by a little window informing you of the successful completion, please press 'OK' to finish the upload process.
Logging In

Once you have followed all the necessary steps described in the sections 'What to do before you use the e-Minerals Mini Grid ..' you can now login to the protected part of the grid under:

If you have not loaded your certificate into a MyProxy server you will need to do so now by clicking on: 'e-Minerals Proxy' at the bottom of the page.

You will be asked for:

User name - the name with which you registered in the e-Minerals Mini Grid and which you should also always use when uploading your certificate onto the MyProxy server.

Passphrase - passphrase given when you loaded your certificate into the MyProxy Server

Lifetime - the lifetime of the credentials. Basically the length of time you wish to be active of the DataPortal. Usually 2 hours.

You will then be presented with the Baisc Search page.
