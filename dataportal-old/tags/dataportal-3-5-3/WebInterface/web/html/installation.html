Data Portal proxy tool installation guide



Web Start Instructions

1) Download and install Java Web Start
2) Download Bouncy Castle jar file and put it into your <java_home>/jre/lib/ext directory, where <java_home> represents which <java_home> the Web Start Application is using.
This can be different to the <java_home> you have set as a envirnmental variable.

To find the <java_home> which web start is using first open the Web Start Application Manager, this can be noramlly found on your desktop or in you Start/Programs.

Click on File/Preferences and click java. This will show which JREs you have installed and which one Web Start uses (The one enabled).

Go to the directory above the bin directory in the Command field of the JRE which is enabled.
Then download Bouncy Castle jar file into the jre/lib/ext or the lib/ext directory.

You are know ready to use the Proxy Tool.

Tar and Zip file instructions

1) Download and unpack the file. Execute /run.sh or ./run.bat depending on your OS. README has more information.
You are know ready to use the Proxy Tool.
