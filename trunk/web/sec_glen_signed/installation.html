Installation of MyProxy Upload Tool

In order for the Java WebStart client to verify the signature of the jars it needs to know the CA that issued the signing certificate to the entity that signed the jars. That means that before using any of the applications a user needs to import the UK e-Science and Sun's "JCE Code Signing CA" certificate authorities into their Java WebStart client.

Download the CA certificates:
UK e-Science CA
JCE Code Signing CA

You will also have to import my certificate as a trusted certificate into WebStart

Download my certificate:
Glen Drinkwater

Here are the steps to install the CA certificates into Java WebStart:

Start Java WebStart console
Go to Edit->Perferences
Go to Security -> Certificates
Click on Signer CA
Click import and then select the two files containing the CA Certificates

Here are the steps to install the my certificate into Java WebStart:

Start Java WebStart console
Go to Edit->Perferences
Go to Security -> Certificates
Click on Trusted Cerificates
Click import and select the file containing my certificate

Only then will Java WebStart be able to validate the signature of the jars. Since the 'Legion of the Bouncy Castle' is signed by Sun's JCE Code Signing CA you can trust this by pressing 'Always' once you have launched the tool.

Launch MyProxy Upload Tool.
