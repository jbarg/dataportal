MyProxy Upload Tool





If you only have Java 1.4 installed the MyProxy Upload Tool will not work with properly signed certificates, like the one from the Uk e-Science CA. This is due to the version of Web Start bundled with Java 1.4. Java's 1.5 Web Start fixes this problem and can be installed with correct certificates.
If you trust this site and the MyProxy Upload Tool been signed by a self signed certificate by Glen Drinkwater then Launch MyProxy Upload Tool.
If not, download and install Java 1.5 and go though the installation guide.
