Welcome to the CCLRC DataPortal




This is a test system and will be updated frequently to integrate and test new features, please bear with us.


The DataPortal currently allows access to selected metadata and data from four facilities. The first three housed by CLRC:
  • The Synchrotron Radiation Department (SRD)
  • The Neutron Spallation Source (ISIS)
  • The British Atmospheric Data Centre (BADC)
  • Max-Planck Institute for Meteorology (MPIM)

  • Login
Please note that the DataPortal uses Globus certificates for authentication, contact the dataportal team for more information.  If you wish to use are old version of the Data Portal (no longer updated) it can still be accessed under old CCLRC Data Portal without the use of certificates, a test account is available.

The DataPortal uses none persistent cookies and JavaScript and therefore these need to be turn on for access.
The web site also uses web standards XHTML 1.0, CSS 1.0 and JavaScript 1.3. Because of this it is best viewed using a browser that complies with these, e.g. Internet Explorer 5.x/6.x, Netscape 6.x or Opera 6.x. For more information on web standards visit the Web Standards project
