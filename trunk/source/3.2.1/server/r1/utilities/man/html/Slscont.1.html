NAME

Slscont - list your SRB containers  

SYNOPSIS

Slscont [-a] [-l] [-z mcatZone]
Slscont container [-z mcatZone]  

DESCRIPTION

In the first synopsis form, Slscont displays all containers owned by this user. The -a option allows the display of all containers accessible by this user.

In the second synopsis form, all inContainer objects contained in the given SRB container will be listed. The container may be an absolute path or a relative path in the SRB name space.

 

OPTIONS

-h
display command options
-l
display information in long form.
-a
display containers accessible by the user rather than owned by the user.
-z mcatZone
The MCAT zone to query for the container info
