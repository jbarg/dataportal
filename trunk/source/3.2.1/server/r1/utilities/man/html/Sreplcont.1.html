NAME

Sreplcont - Replicate a container copy.  

SYNOPSIS

Sreplcont [-z mcatZone] [-S resource] container  

DESCRIPTION

Sreplcont Replicates a container copy to a specific resource. For containers that have multiple "permanent" and "cache" copies, this is a way to put a copy of the container on a specific resource. For example, if a container was created in multiple "cache" resources, this is a way to stage this container on a "cache" resource that is not designated as "primary". The specified resource must be one of the physical resources used to recreate the container (i.e., the specified resource must belong to the logical resource used to create the container).

 

OPTIONS

-h
display command options
-z mcatZone
The MCAT zone where the container resides
-S
resource the resource where the container copy is to be created.
