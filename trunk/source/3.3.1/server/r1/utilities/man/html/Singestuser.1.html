NAME

Singestuser - adds an account to a SRB grid.  

SYNOPSIS

Singestuser NewUserName NewUserPassword NewUserDomain NewUserType NewUserAddress NewUserPhone NewUserEmail NewUserAuthScheme NewUserDNString [ zoneName ]  

DESCRIPTION

Singestuser adds an account to a SRB grid. Be careful to quote fields with spaces.

 

FIELDS

NewUserName
the name for the account
NewUserPassword
the account password
NewUserDomain
the domain of the account
NewUserType
the type of account (staff, sysadmin, etc...) use "Stoken UserType"
NewUserAddress
the address of the user
NewUserPhone
the phone number of the user
NewUserEmail
the email of the user
NewUserAuthScheme
how the user will be authenticated (ENCRYPT1, GSI_AUTH, etc...) (by default not enforced)
NewUserDNString
Distinguished Name for GSI authentication
ZoneName
name of the foreign zone to which the user belongs. If not given, the user will be autmatically registered to the local zone.
 

SEE ALSO

SmodifyUser(1), SgetU(1), Stoken(1)
