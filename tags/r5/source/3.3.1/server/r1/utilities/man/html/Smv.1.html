NAME

Smv - changes the collection for objects in SRB space  

SYNOPSIS

Smv srbObj targetObj
Smv collection newcollection
Smv [-p] srbObj ... collection  

DESCRIPTION

In the first synopsis, Smv moves the SRB object named srbObj into the destination specified by targetObj. The targetObj should not exist; otherwise, the operation is not performed.

In the second synopsis, Smv moves a SRB collection named collection to a new destination collection called newcollection.

In the third synopsis, Smv moves each srbObj to a new collection.

Smv does not move the SRB objects physically. If any srbObj is replicated then every copy is moved to the new location. The user should have 'all' access permission for each SrbObj.

 

OPTIONS

-h
display command options
-p
prompt for input srbObj
