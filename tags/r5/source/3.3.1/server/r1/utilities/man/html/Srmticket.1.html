NAME

Srmticket - remove a ticket  

SYNOPSIS

Srmticket [-F fileName] | ticket  

DESCRIPTION

Srmticket removes a previously issued ticket. One has to own the ticket to remove it.

 

OPTIONS

-h
display command options
-F
fileName specifies the file in which the ticket is stored.

Sticket(1), Stls(1), Stcat(1)
