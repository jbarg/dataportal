SDSC MATRIX


SDSC Matrix provides a standards based approach to deliver Inter-organizational Information Storage Management services, for building datagrids, digital libraries and persistent archives.

 

SDSC Matrix is a protocol layer on top of the SRB/MCAT protocol stack providing value-addition in terms of interoperability, asynchronous messages and stateful services.

 

This is the first of a series of releases for SDSC Matrix, as it is under active development.
