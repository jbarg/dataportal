NAME

Schdefres - change default resource for the current session at the terminal.  

SYNOPSIS

Schdefres resourceName  

DESCRIPTION

Schdefres changes the default resource used in commands such as Sput, Sbload, etc.

Note that reissuing an Sinit command will revert back to the default resource in .MdasEnv file. Sessions in other terminals are not affected.

 

OPTIONS

none  

SEE ALSO

Sinit(1), Senv(1) Schhost(1) Sput(1)
