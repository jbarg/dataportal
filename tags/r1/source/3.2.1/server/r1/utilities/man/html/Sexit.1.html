NAME

Sexit - Ends a SRB session  

SYNOPSIS

Sexit  

DESCRIPTION

Sexit routine ends a SRB session started with Sinit. It cleans up any non-persistent environment files created during the session. User should have write permissions in the path where the Client Environment files are located.

If not invoked, it will result in having several session Client Environment files unfreed.

Recommended to have it as part of logout script.

 

OPTIONS

none
