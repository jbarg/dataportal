The Definition Of Participant Collaboration

The following charts represent the number of collaborations between sites and within a site.
A collaboration is defined as a participant accessing (read, copy, view, modify) a file other than
his or her own as reported by the SRB audit logs. We are providing both accumulative collaboration counts
as well as the current monthly counts. Average file size accessed in MB is also shown.
