XQuery Implementation


Functionalities
1) Parse For Clause
	- Generate File System / File depending on query type
	
2) Parse Where Clause
	- Parse Conditional Clauses
	- Parse Condtion Variable , Operator and Condition Value
	- Create User Defined Meta Data Table

3) Parse Return Clause
	- Parse Selection Fields
	- Mapping of Selection Fields to Query Result	

4) Map Matrix Variables to SRB Variables
	- Map Matrix Variable to the list of 100 or so variables
	- Distinguish User Defined Meta Data

5) Format Query Result
	- Reiterate the formatting for the query result

6) Exceptions
	- No Query Results
	- Wrong Query Request
	- Indicate Not Supported Functionality
