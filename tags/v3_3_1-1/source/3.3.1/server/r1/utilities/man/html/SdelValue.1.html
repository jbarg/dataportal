NAME

SdelValue - Deletes a SRB entry.  

SYNOPSIS

SdelValue <operationType> DeleteValue  

DESCRIPTION

SdelValue deletes a SRB entry. <operationType> determines which type of entry is to be deleted. DeleteValue is the entry you wish to delete.

 

OPERATIONTYPES

User
Deletes a user.
PhyResource
Deletes a physical resource.
Location
Deletes a location.
